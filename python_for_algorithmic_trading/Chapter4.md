# 储存格式对比

| 场景                    | 建议格式                |
| --------------------- | ------------------- |
| 历史 K 线、tick、分钟线等大规模时序 | **HDF5**（或 Parquet） |
| 多标的因子矩阵 / 模型训练输入      | **HDF5**            |
| 回测结果存档、统计表格           | **Parquet**         |
| 数据接口缓存 / 配置表          | **SQLite**          |
| 导入导出（Excel交互）         | **CSV**             |
